## Installation

### Homebrew (macOS/Linux)

```bash
brew install vijaypotnuru/tap/dispatch
```

### Build from Source

```bash
git clone https://github.com/vijaypotnuru/dispatch.git
cd dispatch
make build
cp server/bin/dispatch /usr/local/bin/dispatch
```

### Download from GitHub Releases

If Homebrew is not available, download the binary directly:

```bash
OS=$(uname -s | tr '[:upper:]' '[:lower:]')   # "darwin" or "linux"
ARCH=$(uname -m)                                # "x86_64" or "arm64"

# Normalize architecture name
if [ "$ARCH" = "x86_64" ]; then
  ARCH="amd64"
fi

# Get the latest release tag from GitHub
LATEST=$(curl -sI https://github.com/vijaypotnuru/dispatch/releases/latest \
  | grep -i '^location:' | sed 's/.*tag\///' | tr -d '\r\n')

# Download and extract
curl -sL "https://github.com/vijaypotnuru/dispatch/releases/download/${LATEST}/dispatch_${OS}_${ARCH}.tar.gz" \
  -o /tmp/dispatch.tar.gz
tar -xzf /tmp/dispatch.tar.gz -C /tmp dispatch
sudo mv /tmp/dispatch /usr/local/bin/dispatch
rm /tmp/dispatch.tar.gz
```

### Update

```bash
brew upgrade vijaypotnuru/tap/dispatch
```

For install script or manual installs, use:

```bash
dispatch update
```

`dispatch update` auto-detects your installation method and upgrades accordingly.

## Quick Start

```bash
# One command: configure, authenticate, and start the daemon
dispatch setup
```

This configures the CLI for Dispatch Cloud, opens your browser for login, discovers your workspaces, and starts the agent daemon.

For self-hosted servers, use `dispatch setup self-host` instead. See [Self-Hosting](/getting-started/self-hosting) for details.

## Verify

```bash
dispatch daemon status
```

Confirm:
1. Status is `running`
2. At least one agent is listed (e.g. `claude`, `codex`, `gemini`, `opencode`, `openclaw`, `hermes`, or `pi`)
3. At least one workspace is being watched

If the agents list is empty, install at least one supported AI agent CLI:
- [Claude Code](https://docs.anthropic.com/en/docs/claude-code) (`claude`)
- [Codex](https://github.com/openai/codex) (`codex`)
- [Gemini CLI](https://github.com/google-gemini/gemini-cli) (`gemini`)
- OpenCode (`opencode`)
- OpenClaw (`openclaw`)
- Hermes (`hermes`)

Then restart the daemon:

```bash
dispatch daemon stop && dispatch daemon start
```
